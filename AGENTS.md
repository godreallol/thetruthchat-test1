## Change Control
- Do not modify any file unless I explicitly say: "apply the change".
- Treat only the exact phrase "apply the change" as permission to edit files; phrases like "do it", "go ahead", or similar are not sufficient.
- For every request, default to analysis/explanation only.
- Before any command that edits files, ask for confirmation in one short sentence.
- If unclear whether I want edits, assume I do not.
